# Windows 手动打包说明

## 问题
Linux 环境无法直接打包 Windows 应用（缺少 wine），需要在 Windows 上执行打包。

## 快速打包步骤

### 方法 1: 使用批处理脚本（推荐）

1. 打开 CMD（按 `Win + R`，输入 `cmd` 回车）
2. 输入以下命令：
   ```cmd
   cd "C:\Users\liu_j\Desktop\SJTU\AI\Vibe coding\research\canvas-drop-submit"
   rebuild-windows.bat
   ```
3. 等待打包完成

### 方法 2: 手动执行

1. 删除旧的打包文件：
   ```cmd
   cd "C:\Users\liu_j\Desktop\SJTU\AI\Vibe coding\research\canvas-drop-submit"
   rmdir /s /q dist
   ```

2. 构建项目：
   ```cmd
   npm run build
   ```

3. 打包 Windows 应用：
   ```cmd
   npm run build:win
   ```

## 打包输出

打包完成后，文件位置：
- **绿色版**: `dist\win-unpacked\Canvas Drop Submit.exe`
- **安装程序**: `dist\Canvas Drop Submit Setup.exe`

## 注意事项

1. 确保已经安装了 Node.js 和 npm
2. 如果提示缺少依赖，先运行 `npm install`
3. 打包过程可能需要几分钟，请耐心等待

## 验证打包结果

打包完成后，可以运行以下命令验证：
```cmd
dir "dist\win-unpacked\Canvas Drop Submit.exe"
```

如果文件大小约为 170MB，说明打包成功。
